# 📜 CAPÍTULO I

## 1.1 ORDEN PÚBLICO (PUBLIC ORDER) – FACTORES MATEMÁTICOS

El **Orden Público (PO)** no es un valor abstracto. Es la suma directa de modificadores positivos y negativos expresados en porcentaje.

La fórmula interna simplificada es:

```
Orden Público (%) = 100 
+ Bonificadores 
- Penalizadores
```

Si el resultado final:

* ≥ 100% → Ciudad estable
* 70–99% → Riesgo bajo/moderado
* < 70% → Alto riesgo de rebelión
* < 50% → Rebelión casi garantizada

# 🧮 FACTORES MATEMÁTICOS DETALLADOS

Voy a desglosarlo como lo hace el motor del juego.

---

## I. BASE

### 🔹 1. Base inicial

Siempre partes de:

```
100%
```

Es el punto de equilibrio neutral.

---

## II. BONIFICADORES (SUMAN %)

---

### 🏛 1. Leyes (Law)

Proviene de:

* Edificios administrativos (Foro, Basílica, etc.)
* Templos que den Law
* Rasgos del gobernador
* Seguidores

Ejemplo matemático:

| Fuente                   | Valor |
| ------------------------ | ----- |
| Basílica                 | +10%  |
| Templo de ley alto       | +15%  |
| Rasgo buen administrador | +5%   |

**Total Leyes: +30%**

---

### 👑 2. Lealtad / Influencia del Gobernador

Depende de:

* Autoridad
* Rasgos
* Seguidores

Cada punto efectivo suele aportar aproximadamente:

```
+0.5% a +1% por punto real
```

Un gobernador fuerte puede dar:

```
+10% a +25%
```

---

### ⚔ 3. Presencia Militar

La fórmula aproximada es:

```
Unidades en ciudad × valor de represión
```

No todas las tropas valen igual.

| Tipo de unidad    | Valor aproximado |
| ----------------- | ---------------- |
| Milicia básica    | 2–4%             |
| Infantería pesada | 4–6%             |
| Élite             | 6–8%             |

Ejemplo:
4 cohortes urbanas = 4 × 6% = +24%

⚠ El efecto tiene rendimiento decreciente.

---

### 🎭 4. Entretenimiento

Proporcionado por:

* Anfiteatro
* Arena
* Coliseo

Puede sumar:

```
+5% a +20%
```

---

### 🛕 5. Religión (si cultura coincide)

Si tu templo es de tu cultura:
+bono oculto de estabilidad.

---

### 😊 6. Felicidad por baja población

Ciudades pequeñas suelen tener menos penalizador demográfico.

---

## III. PENALIZADORES (RESTAN %)

---

### 💰 1. Impuestos

Uno de los factores más importantes.

| Nivel     | Penalización |
| --------- | ------------ |
| Muy bajos | 0%           |
| Bajos     | -5%          |
| Normales  | -10%         |
| Altos     | -20%         |
| Muy altos | -30%         |

Esto es fijo.

---

### 🏙 2. Tamaño de Población (Factor crítico)

Cada umbral de ciudad añade presión:

| Nivel          | Penalización base |
| -------------- | ----------------- |
| Aldea          | 0%                |
| Ciudad pequeña | -5%               |
| Ciudad grande  | -10%              |
| Ciudad enorme  | -15%              |
| Metrópolis     | -20% o más        |

⚠ Este valor escala internamente con población real.

Ciudad de 20.000+ habitantes puede sufrir -25% oculto.

---

### 🧬 3. Diferencia Cultural (Cultural Penalty)

Cuando conquistas otra cultura.

Fórmula aproximada:

```
Penalización inicial: -20% a -30%
Disminuye ~1% por turno
```

Puede tardar 20–30 turnos en desaparecer.

Destruir templos enemigos acelera reducción.

---

### 🧼 4. Inmundicia (Squalor)

Uno de los sistemas más ocultos y peligrosos.

Se calcula según:

```
Población total acumulada históricamente
```

No solo población actual.

Efecto típico:

| Nivel    | Penalización |
| -------- | ------------ |
| Bajo     | -5%          |
| Medio    | -10%         |
| Alto     | -20%         |
| Muy alto | -30%         |

⚠ No desaparece fácilmente.
⚠ Solo se mitiga con baños, acueductos y reducción poblacional (exterminar).

---

### 😡 5. Distancia a Capital

Cuanto más lejos, más corrupción y menos control.

Puede restar:

```
-5% a -25%
```

Imperios extensos sufren mucho aquí.

---

### 🔥 6. Exterminio reciente / Esclavitud

Después de conquistar:

* Ocupación: penalizador leve temporal
* Esclavizar: penalizador medio
* Exterminar: casi ningún penalizador posterior

---

# 📊 EJEMPLO REAL COMPLETO

Ciudad conquistada romana, 12.000 habitantes.

### Bonificadores:

* Leyes: 20%
* Gobernador: 12%
* Tropas: 18%
* Entretenimiento: 10%

Total positivos: +60%

---

### Penalizadores:

* Impuestos normales: -10%
* Tamaño ciudad grande: -10%
* Cultura extranjera: -25%
* Squalor: -15%

Total negativos: -60%

---

### Cálculo:

```
100 + 60 - 60 = 100%
```

Ciudad estable, pero frágil.

---

# ⚙ FACTORES OCULTOS IMPORTANTES

### 1️⃣ Rendimiento decreciente militar

Agregar demasiadas tropas no escala linealmente.

### 2️⃣ Squalor nunca desaparece totalmente

Solo se ralentiza.

### 3️⃣ Cultura extranjera se reduce con:

* Tiempo
* Templos propios
* Destrucción de templos enemigos

---

# 🧠 PRINCIPIOS ESTRATÉGICOS DE VETERANO

✔ La estabilidad verdadera viene de Leyes + Cultura propia.
✔ El ejército es parche temporal.
✔ El impuesto es la palanca económica más peligrosa.
✔ Las metrópolis siempre tenderán a rebelarse si no están especializadas.
✔ Exterminar reduce futuros problemas de squalor.
